Отчёт по декартовой системе координат (ДСК)

1. Исходные данные

Движение точки в неподвижной системе \(Oxy\) задано уравнениями:

\[ \begin{cases} x(t) = a \cdot \cos\left(\dfrac{b\pi t^{2}}{6}\right) \\[6pt] y(t) = -2a \cdot \sin\left(\dfrac{b\pi t^{2}}{6}\right) \\[6pt] t = t_0 = 1\ \text{с} \end{cases} \qquad a = 8,\quad b = -2 \]

Подставляем \(a=8,\ b=-2\):

\[ x(t) = 8 \cos\left(\frac{-2\pi t^{2}}{6}\right) = 8 \cos\left(-\frac{\pi t^{2}}{3}\right) = 8 \cos\left(\frac{\pi t^{2}}{3}\right) \] \[ y(t) = -16 \sin\left(\frac{-2\pi t^{2}}{6}\right) = -16 \sin\left(-\frac{\pi t^{2}}{3}\right) = 16 \sin\left(\frac{\pi t^{2}}{3}\right) \]

Таким образом, закон движения в ДСК:

\[ \boxed{x(t) = 8\cos\left(\frac{\pi t^{2}}{3}\right), \quad y(t) = 16\sin\left(\frac{\pi t^{2}}{3}\right)} \]

Требуется найти скорость, ускорение и радиус кривизны траектории в момент \(t = {{ data.t }}\) с. Построить три интерактивных графика: траектория с положением точки, вектор скорости, вектор ускорения.

2. Положение точки

В момент времени \(t = {{ data.t }}\) с:

\[ x(1) = 8\cos\left(\frac{\pi}{3}\right) = 8 \cdot \frac{1}{2} = 4 \ \text{м} \] \[ y(1) = 16\sin\left(\frac{\pi}{3}\right) = 16 \cdot \frac{\sqrt{3}}{2} = 8\sqrt{3} \approx {{ "%.4f"|format(data.point[1]) }} \ \text{м} \]

Точка \(M\) в момент \(t = {{ data.t }}\) с имеет координаты:

\[ M(4;\ 8\sqrt{3}) \ \text{м} \quad \text{или} \quad M({{ "%.4f"|format(data.point[0]) }};\ {{ "%.4f"|format(data.point[1]) }})\ \text{м} \]

3. Скорость точки

Вектор скорости в декартовой системе координат определяется как:

\[ \vec V = \dot x \,\vec i + \dot y \,\vec j \]

3.1 Компоненты скорости

\[ \dot x(t) = \frac{d}{dt}\left[8\cos\left(\frac{\pi t^2}{3}\right)\right] = 8 \cdot \left[ -\sin\left(\frac{\pi t^2}{3}\right) \cdot \frac{2\pi t}{3} \right] = -\frac{16\pi t}{3} \sin\left(\frac{\pi t^2}{3}\right) \] \[ \dot y(t) = \frac{d}{dt}\left[16\sin\left(\frac{\pi t^2}{3}\right)\right] = 16 \cdot \left[ \cos\left(\frac{\pi t^2}{3}\right) \cdot \frac{2\pi t}{3} \right] = \frac{32\pi t}{3} \cos\left(\frac{\pi t^2}{3}\right) \]

Подставляем \(t = {{ data.t }}\) с:

\[ \dot x(1) = -\frac{16\pi}{3} \sin\left(\frac{\pi}{3}\right) = -\frac{16\pi}{3} \cdot \frac{\sqrt{3}}{2} = -\frac{8\pi\sqrt{3}}{3} \approx {{ "%.4f"|format(data.V[0]) }} \ \text{м/с} \] \[ \dot y(1) = \frac{32\pi}{3} \cos\left(\frac{\pi}{3}\right) = \frac{32\pi}{3} \cdot \frac{1}{2} = \frac{16\pi}{3} \approx {{ "%.4f"|format(data.V[1]) }} \ \text{м/с} \]

3.2 Модуль скорости

\[ V = \sqrt{\dot x^2 + \dot y^2} = \sqrt{\left(-\frac{8\pi\sqrt{3}}{3}\right)^2 + \left(\frac{16\pi}{3}\right)^2} = \frac{8\pi}{3}\sqrt{3 + 4} = \frac{8\pi\sqrt{7}}{3} \approx {{ "%.4f"|format(data.V_mod) }} \ \text{м/с} \]

Вектор скорости в момент \(t = {{ data.t }}\) с:

\[ \vec V = \left({{ "%.4f"|format(data.V[0]) }};\ {{ "%.4f"|format(data.V[1]) }}\right) \ \text{м/с} \]

4. Ускорение точки

Вектор ускорения в декартовой системе координат определяется как:

\[ \vec a = \ddot x \,\vec i + \ddot y \,\vec j \]

4.1 Компоненты ускорения

\[ \ddot x(t) = \frac{d}{dt}\left( -\frac{16\pi t}{3} \sin\left(\frac{\pi t^2}{3}\right) \right) = -\frac{16\pi}{3}\sin\left(\frac{\pi t^2}{3}\right) - \frac{16\pi t}{3}\cos\left(\frac{\pi t^2}{3}\right)\cdot\frac{2\pi t}{3} \] \[ \ddot x(t) = -\frac{16\pi}{3}\sin\left(\frac{\pi t^2}{3}\right) - \frac{32\pi^2 t^2}{9}\cos\left(\frac{\pi t^2}{3}\right) \] \[ \ddot y(t) = \frac{d}{dt}\left( \frac{32\pi t}{3} \cos\left(\frac{\pi t^2}{3}\right) \right) = \frac{32\pi}{3}\cos\left(\frac{\pi t^2}{3}\right) - \frac{32\pi t}{3}\sin\left(\frac{\pi t^2}{3}\right)\cdot\frac{2\pi t}{3} \] \[ \ddot y(t) = \frac{32\pi}{3}\cos\left(\frac{\pi t^2}{3}\right) - \frac{64\pi^2 t^2}{9}\sin\left(\frac{\pi t^2}{3}\right) \]

Подставляем \(t = {{ data.t }}\) с:

\[ \ddot x(1) = -\frac{16\pi}{3}\cdot\frac{\sqrt{3}}{2} - \frac{32\pi^2}{9}\cdot\frac{1}{2} = -\frac{8\pi\sqrt{3}}{3} - \frac{16\pi^2}{9} \approx {{ "%.4f"|format(data.a[0]) }} \ \text{м/с}^2 \] \[ \ddot y(1) = \frac{32\pi}{3}\cdot\frac{1}{2} - \frac{64\pi^2}{9}\cdot\frac{\sqrt{3}}{2} = \frac{16\pi}{3} - \frac{32\pi^2\sqrt{3}}{9} \approx {{ "%.4f"|format(data.a[1]) }} \ \text{м/с}^2 \]

4.2 Модуль ускорения

\[ a = \sqrt{\ddot x^2 + \ddot y^2} \approx \sqrt{({{ "%.4f"|format(data.a[0]) }})^2 + ({{ "%.4f"|format(data.a[1]) }})^2} = {{ "%.4f"|format(data.a_mod) }} \ \text{м/с}^2 \]

Вектор ускорения в момент \(t = {{ data.t }}\) с:

\[ \vec a = \left({{ "%.4f"|format(data.a[0]) }};\ {{ "%.4f"|format(data.a[1]) }}\right) \ \text{м/с}^2 \]
