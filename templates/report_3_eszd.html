Отчёт по естественному способу задания движения (ЕСЗД)

1. Исходные данные

Движение точки в декартовой системе координат задано уравнениями:

\[ x(t) = 8\cos\left(\frac{\pi t^2}{3}\right), \qquad y(t) = 16\sin\left(\frac{\pi t^2}{3}\right) \]

При естественном способе задания движения используются:

  • Дуговая координата \(s(t)\) - расстояние, пройденное точкой вдоль траектории;
  • Скорость \(V(t) = \dot s(t)\) - всегда направлена по касательной к траектории;
  • Касательное ускорение \(a_\tau = \dot V(t)\) - характеризует изменение модуля скорости;
  • Нормальное ускорение \(a_n = \dfrac{V^2}{\rho}\) - характеризует изменение направления скорости;
  • Радиус кривизны \(\rho\) - определяет степень искривления траектории.

2. Кинематические характеристики в естественном виде

2.1 Модуль скорости

\[ V(t) = \sqrt{\dot x^2 + \dot y^2} = \sqrt{\left(-\frac{16\pi t}{3} \sin\frac{\pi t^2}{3}\right)^2 + \left(\frac{32\pi t}{3} \cos\frac{\pi t^2}{3}\right)^2} \] \[ V(t) = \frac{16\pi t}{3} \sqrt{\sin^2\frac{\pi t^2}{3} + 4\cos^2\frac{\pi t^2}{3}} = \frac{16\pi t}{3} \sqrt{1 + 3\cos^2\frac{\pi t^2}{3}} \]

В момент \(t = {{ data.t }}\) с:

\[ V = {{ "%.4f"|format(data.V) }} \ \text{м/с} \]

2.2 Касательное ускорение

Касательное ускорение - производная модуля скорости по времени:

\[ a_\tau = \frac{dV}{dt} = \frac{d}{dt}\sqrt{\dot x^2 + \dot y^2} = \frac{\dot x \ddot x + \dot y \ddot y}{V} \]

В момент \(t = {{ data.t }}\) с:

\[ a_\tau = {{ "%.4f"|format(data.a_tau) }} \ \text{м/с}^2 \]

Примечание: Если \(a_\tau > 0\) - скорость возрастает, если \(a_\tau < 0\) — скорость убывает.

2.3 Нормальное ускорение

Нормальное ускорение направлено к центру кривизны траектории:

\[ a_n = \sqrt{a^2 - a_\tau^2} = \frac{V^2}{\rho} \]

В момент \(t = {{ data.t }}\) с:

\[ a_n = {{ "%.4f"|format(data.a_n) }} \ \text{м/с}^2 \]

2.4 Радиус кривизны траектории

\[ \rho = \frac{V^2}{a_n} = \frac{V^3}{|\dot x \ddot y - \dot y \ddot x|} \]

В момент \(t = {{ data.t }}\) с:

\[ \rho = {{ "%.4f"|format(data.rho) }} \ \text{м} \]

Физический смысл: радиус кривизны показывает, насколько траектория искривлена. Чем меньше \(\rho\), тем сильнее изгиб.

2.5 Дуговая координата

Дуговая координата \(s(t)\) - длина пути, пройденного точкой от начального положения:

\[ s(t) = \int_0^t V(\tau) d\tau \]

В момент \(t = {{ data.t }}\) с:

\[ s = {{ "%.4f"|format(data.s) }} \ \text{м} \]

3. Полное ускорение точки

Полное ускорение точки складывается из касательной и нормальной составляющих:

\[ \vec a = \vec a_\tau + \vec a_n \]

Модуль полного ускорения:

\[ a = \sqrt{a_\tau^2 + a_n^2} = {{ "%.4f"|format(data.a_mod) }} \ \text{м/с}^2 \]
