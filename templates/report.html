Отчёт по сложному движению точки (подробное решение)

Бригада 2, момент времени \( t = {{ data.t }} \)

1. Исходные данные и условия задачи

Нам дана точка, которая движется в подвижной системе координат \(xOy\) по закону:

\[ x(t) = 8\cos\left(\frac{\pi t^2}{3}\right), \qquad y(t) = 16\sin\left(\frac{\pi t^2}{3}\right) \]

Это относительное движение. Одновременно подвижная система \(xOy\) перемещается относительно неподвижной \(x'O'y'\):

\[ z'(t) = 2t^2 + 4, \qquad \varphi'(t) = 2\sin(\pi t) \]

Здесь \(z'(t)\) – смещение начала \(O\) по вертикали (ось \(z'\)), а \(\varphi'(t)\) – угол поворота подвижных осей вокруг вертикальной оси. В момент времени \(t=1\) нужно найти скорость и ускорение точки в абсолютном пространстве, радиус кривизны траектории и построить графики.

2. Относительное движение (в подвижной системе \(xOy\))

Подвижная система вращается и движется поступательно, но в ней точка имеет координаты, заданные выше. Найдём скорость и ускорение в относительном движении – производные по времени.

Скорость точки относительно подвижной системы

\[ \dot x = -\frac{16\pi t}{3} \sin\left(\frac{\pi t^2}{3}\right),\quad \dot y = \frac{32\pi t}{3} \cos\left(\frac{\pi t^2}{3}\right) \] При \( t = {{ data.t }} \): \[ \dot x = {{ "%.4f"|format(data.V_rel[0]) }},\quad \dot y = {{ "%.4f"|format(data.V_rel[1]) }},\quad V_{\text{отн}} = {{ "%.4f"|format(data.V_rel_mod) }} \ \text{м/с} \]

Ускорение точки относительно подвижной системы

\[ \ddot x = -\frac{16\pi}{3}\sin\left(\frac{\pi t^2}{3}\right) - \frac{32\pi^2 t^2}{9}\cos\left(\frac{\pi t^2}{3}\right) \] \[ \ddot y = \frac{32\pi}{3}\cos\left(\frac{\pi t^2}{3}\right) - \frac{64\pi^2 t^2}{9}\sin\left(\frac{\pi t^2}{3}\right) \] При \( t = {{ data.t }} \): \[ \ddot x = {{ "%.4f"|format(data.a_rel[0]) }},\quad \ddot y = {{ "%.4f"|format(data.a_rel[1]) }} \]

3. Переносное движение (движение подвижной системы)

Подвижная система движется двумя способами: поступательно (вертикальное смещение) и вращательно (поворот вокруг вертикальной оси).

Поступательная часть

\[ z'(t) = 2t^2 + 4,\quad \dot z'(t) = 4t,\quad \ddot z'(t) = 4 \] При \( t = {{ data.t }} \): \[ \dot z' = {{ "%.4f"|format(data.dzpdt) }},\quad \ddot z' = {{ "%.4f"|format(data.d2zpdt2) }} \] Скорость поступательного переноса: \(\mathbf{V}_{\text{пер,пост}} = (0,0,{{ "%.4f"|format(data.dzpdt) }})\).
Ускорение: \(\mathbf{a}_{\text{пер,пост}} = (0,0,{{ "%.4f"|format(data.d2zpdt2) }})\).

Вращательная часть

\[ \varphi'(t) = 2\sin(\pi t),\quad \omega = \dot\varphi' = 2\pi\cos(\pi t),\quad \alpha = \ddot\varphi' = -2\pi^2\sin(\pi t) \] При \( t = {{ data.t }} \): \[ \varphi' = {{ "%.4f"|format(data.phi) }},\quad \omega = {{ "%.4f"|format(data.omega) }},\quad \alpha = {{ "%.4f"|format(data.alpha) }} \] Вектор угловой скорости: \(\boldsymbol{\omega} = (0,0,{{ "%.4f"|format(data.omega) }})\).

Радиус-вектор точки \(M\) в подвижной системе в момент \(t=1\): \(\mathbf{r}_{\text{отн}} = ({{ "%.4f"|format(data.V_rel[0]) }}, {{ "%.4f"|format(data.V_rel[1]) }}, 0)\).

Переносная скорость (вращательная)

\[ \mathbf{V}_{\text{пер,вр}} = \boldsymbol{\omega} \times \mathbf{r}_{\text{отн}} = ({{ "%.4f"|format(data.V_rot[0]) }}, {{ "%.4f"|format(data.V_rot[1]) }}, 0) \] Модуль: \({{ "%.4f"|format(data.V_rot_mod) }}\) м/с.

Переносное ускорение

Центростремительное: \(\mathbf{a}_{\text{пер,ц}} = -\omega^2 \mathbf{r}_{\text{отн}} = ({{ "%.4f"|format(data.a_centr[0]) }}, {{ "%.4f"|format(data.a_centr[1]) }}, 0)\)
Вращательное: \(\mathbf{a}_{\text{пер,вр}} = \boldsymbol{\alpha} \times \mathbf{r}_{\text{отн}} = ({{ "%.4f"|format(data.a_rot[0]) }}, {{ "%.4f"|format(data.a_rot[1]) }}, {{ "%.4f"|format(data.a_rot[2]) }})\)
Поступательное: \(\mathbf{a}_{\text{пер,пост}} = (0,0,{{ "%.4f"|format(data.d2zpdt2) }})\)
Суммируем: \(\mathbf{a}_{\text{пер}} = ({{ "%.4f"|format(data.a_centr[0]) }}, {{ "%.4f"|format(data.a_centr[1]) }}, {{ "%.4f"|format(data.d2zpdt2) }})\)

4. Сложное движение (абсолютные скорость и ускорение)

Теорема сложения скоростей: \(\mathbf{V}_{\text{абс}} = \mathbf{V}_{\text{отн}} + \mathbf{V}_{\text{пер,пост}} + \mathbf{V}_{\text{пер,вр}}\).

Теорема сложения ускорений: \(\mathbf{a}_{\text{абс}} = \mathbf{a}_{\text{отн}} + \mathbf{a}_{\text{пер}} + \mathbf{a}_{\text{кор}}\), где \(\mathbf{a}_{\text{кор}} = 2\,\boldsymbol{\omega}\times\mathbf{V}_{\text{отн}}\).

Абсолютная скорость

\[ \mathbf{V}_{\text{абс}} = ({{ "%.4f"|format(data.V_abs[0]) }}, {{ "%.4f"|format(data.V_abs[1]) }}, {{ "%.4f"|format(data.V_abs[2]) }}) \] Модуль: \({{ "%.4f"|format(data.V_abs_mod) }}\) м/с.

Ускорение Кориолиса

\[ \mathbf{a}_{\text{кор}} = ({{ "%.4f"|format(data.a_cor[0]) }}, {{ "%.4f"|format(data.a_cor[1]) }}, 0) \] Модуль: \({{ "%.4f"|format(data.a_cor_mod) }}\) м/с².

Абсолютное ускорение

\[ \mathbf{a}_{\text{абс}} = ({{ "%.4f"|format(data.a_abs[0]) }}, {{ "%.4f"|format(data.a_abs[1]) }}, {{ "%.4f"|format(data.a_abs[2]) }}) \] Модуль: \({{ "%.4f"|format(data.a_abs_mod) }}\) м/с².

Радиус кривизны абсолютной траектории

\[ \rho = \frac{V_{\text{абс}}^3}{|\mathbf{V}_{\text{абс}} \times \mathbf{a}_{\text{абс}}|} \approx {{ "%.4f"|format(data.rho) }} \ \text{м} \]

5. Интерактивные 3D-графики

Рис. 1. Абсолютная траектория точки (неподвижная система \(X',Y',Z'\))

Красные оси – подвижная система \(X,Y,Z\) из точки \(O\). Зелёная дуга – направление вращения (угловая скорость \(\omega\)). Фиолетовый вектор – абсолютная скорость (масштабирован для наглядности).

Рис. 2. Векторы скоростей в точке \(M\) (оси: \(V_{\text{отн}},\ a_{\text{cor}},\ \omega\))

Все векторы пересчитаны в систему координат, где оси направлены вдоль относительной скорости, ускорения Кориолиса и угловой скорости.

Рис. 3. Векторы ускорений в точке \(M\) (оси: \(\omega,\ a_{\text{cor}},\ a_{\text{отн}}\))

Аналогично, ускорения показаны в системе, где оси совпадают с направлениями \(\omega\), \(a_{\text{cor}}\) и перпендикулярной компонентой \(a_{\text{отн}}\).
