Отчёт по полярной системе координат (ПСК)

1. Исходные данные и условия задачи

Движение точки задано в декартовой системе координат уравнениями:

\[ x(t) = 8\cos\left(\frac{\pi t^2}{3}\right), \qquad y(t) = 16\sin\left(\frac{\pi t^2}{3}\right) \]

Требуется перейти к полярной системе координат \((r, \varphi)\) и найти:

  • положение точки в ПСК (\(r\) и \(\varphi\));
  • радиальную (\(V_r\)) и трансверсальную (\(V_\varphi\)) составляющие скорости;
  • радиальную (\(a_r\)) и трансверсальную (\(a_\varphi\)) составляющие ускорения;
  • радиус кривизны траектории \(\rho\);
  • построить интерактивные графики.

Связь между декартовыми и полярными координатами:

\[ r = \sqrt{x^2 + y^2}, \qquad \varphi = \operatorname{arctg}\frac{y}{x} \]

2. Переход к полярным координатам

В момент времени \(t = {{ data.t }}\):

\[ x(1) = 8\cos\left(\frac{\pi}{3}\right) = 8 \cdot \frac{1}{2} = 4 \ \text{м} \] \[ y(1) = 16\sin\left(\frac{\pi}{3}\right) = 16 \cdot \frac{\sqrt{3}}{2} = 8\sqrt{3} \approx 13.8564 \ \text{м} \]

Радиальная координата \(r\)

\[ r = \sqrt{x^2 + y^2} = \sqrt{4^2 + (8\sqrt{3})^2} = \sqrt{16 + 192} = \sqrt{208} = 4\sqrt{13} \approx {{ "%.4f"|format(data.r) }} \ \text{м} \]

Угловая координата \(\varphi\)

\[ \varphi = \operatorname{arctg}\frac{y}{x} = \operatorname{arctg}\frac{8\sqrt{3}}{4} = \operatorname{arctg}(2\sqrt{3}) \approx {{ "%.4f"|format(data.phi) }} \ \text{рад} \ ({{ "%.1f"|format(data.phi * 180 / 3.14159) }}^\circ) \]

3. Скорость точки в полярных координатах

В полярной системе координат вектор скорости раскладывается на две составляющие:

\[ \mathbf{V} = V_r \mathbf{e}_r + V_\varphi \mathbf{e}_\varphi \]

где:

\[ V_r = \dot r \quad \text{— радиальная составляющая (скорость изменения расстояния)}, \] \[ V_\varphi = r \dot \varphi \quad \text{— трансверсальная составляющая (переносная скорость)}. \]

Производные полярных координат по времени

В момент времени \(t = {{ data.t }}\):

\[ \dot r = \frac{dr}{dt} \approx {{ "%.4f"|format(data.dr_dt) }} \ \text{м/с} \] \[ \dot \varphi = \frac{d\varphi}{dt} \approx {{ "%.4f"|format(data.dphi_dt) }} \ \text{рад/с} \]

Составляющие скорости

Радиальная скорость:

\[ V_r = \dot r = {{ "%.4f"|format(data.Vr) }} \ \text{м/с} \]

Трансверсальная скорость:

\[ V_\varphi = r \dot \varphi = {{ "%.4f"|format(data.r) }} \cdot {{ "%.4f"|format(data.dphi_dt) }} = {{ "%.4f"|format(data.Vphi) }} \ \text{м/с} \]

Модуль скорости:

\[ V = \sqrt{V_r^2 + V_\varphi^2} = \sqrt{({{ "%.4f"|format(data.Vr) }})^2 + ({{ "%.4f"|format(data.Vphi) }})^2} = {{ "%.4f"|format(data.V_mod) }} \ \text{м/с} \]

4. Ускорение точки в полярных координатах

В полярной системе координат ускорение раскладывается на радиальную и трансверсальную составляющие:

\[ \mathbf{a} = a_r \mathbf{e}_r + a_\varphi \mathbf{e}_\varphi \]

где:

\[ a_r = \ddot r - r \dot \varphi^2 \quad \text{— радиальная составляющая}, \] \[ a_\varphi = r \ddot \varphi + 2 \dot r \dot \varphi \quad \text{— трансверсальная составляющая}. \]

Вторые производные

В момент времени \(t = {{ data.t }}\):

\[ \ddot r = \frac{d^2r}{dt^2} \approx {{ "%.4f"|format(data.d2r_dt2) }} \ \text{м/с}^2 \] \[ \ddot \varphi = \frac{d^2\varphi}{dt^2} \approx {{ "%.4f"|format(data.d2phi_dt2) }} \ \text{рад/с}^2 \]

Составляющие ускорения

Радиальная составляющая:

\[ a_r = \ddot r - r \dot \varphi^2 = {{ "%.4f"|format(data.d2r_dt2) }} - {{ "%.4f"|format(data.r) }} \cdot ({{ "%.4f"|format(data.dphi_dt) }})^2 = {{ "%.4f"|format(data.ar) }} \ \text{м/с}^2 \]

Трансверсальная составляющая:

\[ a_\varphi = r \ddot \varphi + 2 \dot r \dot \varphi = {{ "%.4f"|format(data.r) }} \cdot {{ "%.4f"|format(data.d2phi_dt2) }} + 2 \cdot {{ "%.4f"|format(data.dr_dt) }} \cdot {{ "%.4f"|format(data.dphi_dt) }} = {{ "%.4f"|format(data.aphi) }} \ \text{м/с}^2 \]

Модуль ускорения:

\[ a = \sqrt{a_r^2 + a_\varphi^2} = \sqrt{({{ "%.4f"|format(data.ar) }})^2 + ({{ "%.4f"|format(data.aphi) }})^2} = {{ "%.4f"|format(data.a_mod) }} \ \text{м/с}^2 \]

Примечание: Радиальная составляющая включает центростремительное ускорение \(-r\dot\varphi^2\). Трансверсальная составляющая включает кориолисово ускорение \(2\dot r \dot\varphi\).

5. Радиус кривизны траектории

Радиус кривизны плоской кривой можно вычислить через компоненты скорости и ускорения в полярных координатах:

\[ \rho = \frac{V^3}{|V_r a_\varphi - V_\varphi a_r|} \]

Подставляем полученные значения:

\[ \rho = \frac{({{ "%.4f"|format(data.V_mod) }})^3}{|{{ "%.4f"|format(data.Vr) }} \cdot {{ "%.4f"|format(data.aphi) }} - {{ "%.4f"|format(data.Vphi) }} \cdot {{ "%.4f"|format(data.ar) }}|} \approx {{ "%.4f"|format(data.rho) }} \ \text{м} \]

Это значение показывает, насколько траектория искривлена: чем меньше радиус, тем сильнее изгиб.

6. Графики

Рис. 1. Траектория движения точки

Показана траектория движения точки (синяя линия), радиус-вектор из начала координат в точку M (фиолетовый пунктир), дуга, обозначающая угол \(\varphi\), и сама точка M в момент времени \(t = {{ data.t }}\).

Рис. 2. Радиальная и трансверсальная составляющие скорости

Показаны: радиус-вектор (фиолетовый), радиальная составляющая скорости \(V_r\) (синяя, направлена вдоль радиус-вектора), трансверсальная составляющая \(V_\varphi\) (оранжевая, перпендикулярна радиус-вектору), результирующий вектор скорости \(V\) (зелёный). Серые пунктиры иллюстрируют правило параллелограмма.

Рис. 3. Радиальная и трансверсальная составляющие ускорения

Показаны: радиус-вектор (фиолетовый), радиальная составляющая ускорения \(a_r\) (синяя), трансверсальная составляющая \(a_\varphi\) (оранжевая), результирующий вектор ускорения \(a\) (зелёный).
